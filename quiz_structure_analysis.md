# Love Type 16 測驗網站分析

## 測驗流程

1.  **進入頁面**: 直接開始測驗，顯示第一題。
2.  **題目形式**: 
    *   每頁顯示數道題目。
    *   每題都是「Yes/No」的是非題。
    *   使用者透過點擊代表「Yes」或「No」的圖示來回答。
3.  **進度顯示**: 頁面頂部有進度條，顯示目前完成的百分比。
4.  **分頁**: 測驗分為多頁，每頁包含數道題目，使用者需點擊「Next Page」按鈕進入下一頁。
5.  **完成測驗**: 完成所有題目後，網站會分析答案並導向結果頁面。

## 網站架構與設計

*   **導覽列**: 包含「Love Diagnosis Test」、「About Love Type 16」、「Philosophy」、「Character Introduction」、「Contact」等連結。
*   **語言切換**: 提供多語言切換功能。
*   **美術風格**: 卡通、可愛風格，與我們的目標（維多利亞黑暗風格）不同。
*   **互動設計**: 使用心形圖示作為選項，點擊後會顯示下一個問題或進入下一頁。

3.  **結果計算**: 根據使用者在四個維度上的傾向，組合出 16 種 BDSM 人格類型（例如：DSBP, sMFH 等）。
4.  **結果呈現**: 為每種人格類型撰寫詳細的分析，包括：

    *   核心特質與慾望
    *   模擬的個人經歷或內心獨白
    *   在 BDSM 關係中的角色與行為模式
    *   最互補的伴侶類型分析

    *   網站整體採用黑暗色調，搭配維多利亞時期的華麗、神秘元素。
    *   為 16 種人格類型設計對應的 18+ 維多利亞風格人物插圖。
6.  **多語言支援**: 所有內容（測驗題目、類型分析、網站介面）都需要翻譯成繁體中文、簡體中文和英文三種版本。
